# Roles (Role Packs) — SSOT

本目录提供“角色包（Role Pack）”的权威落点：拿到某个角色包=成为该角色。

规则：
- 角色包只定义：使命/边界、输入输出、允许技能、停止条件、最小流程。
- 角色的“长期记忆”不写在 SSOT；记忆作为可更新输入，落在 `docs/INPUTS/role_memory/`，角色包只做引用。
