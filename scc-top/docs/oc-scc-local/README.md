# Docs

本目录已收敛为**单页索引入口**，请从 `docs/INDEX.md` 进入。

- SCC（scc-top）文档的唯一权威入口：`C:\scc\scc-top\docs\START_HERE.md`
- 总目标（Mission）：`docs/MISSION.md`

- `docs/STATUS.md`：当前执行状态（端口、进度、已知失败原因）
- `docs/RUNBOOK.md`：本地启动/停止/冒烟检查
- `docs/EXECUTOR.md`：队长并行执行器（接口、日志、监控、原子任务）
- `docs/PROMPTING.md`：给 CLI 的任务拆分与提示词规范（不要太大/不要太小）
- `docs/INDEX.md`：唯一入口（运行时必读）
- `docs/AI_CONTEXT.md`：AI 阅读指南（建议上下文、避免大文件）
- `docs/WORKLOG.md`：队长进度流水（运行 `C:\scc\update_worklog.ps1` 自动追加）
- `docs/NAVIGATION.md`：入口与导航整理
