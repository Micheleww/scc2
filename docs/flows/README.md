# Flows (B 模式轻量模板)

目标：在多个小项目并行（B 模式）时，**最少上下文/最短执行路径/一次通过率更高**。

这些模板是“可复制的最小闭环”，适合给 Designer / Factory Manager 生成父任务与原子任务时引用。

## 模板列表

- `docs/flows/flow__feature_patch_v1.md`：最常见的“实现一个小功能”闭环
- `docs/flows/flow__doc_update_v1.md`：文档类任务的闭环（含审计证据要求）
- `docs/flows/flow__ci_fixup_v1.md`：CI 失败后的补救闭环（最多补救 2 次）
