[SCC IO DIGEST v1]

== SUBMIT.JSON REQUIRED FIELDS ==
schema_version: "scc.submit.v1"
task_id: <string>
status: DONE | NEED_INPUT | FAILED
changed_files: [string]
new_files: [string]
allow_paths: { read: [string], write: [string] }
tests: { commands: [string], passed: bool, summary: "..." }
artifacts: { report_md: "...", selftest_log: "...", evidence_dir: "...", patch_diff: "...", submit_json: "..." }
exit_code: 0 (success) | non-zero (failure)
needs_input: [ { id: "...", question: "...", context: "..." } ]

== COMMON FAIL CODES ==
SCOPE_CONFLICT     -> modified files outside pins allowlist
PINS_INSUFFICIENT  -> required file/tool not available in pins
SCHEMA_VIOLATION   -> submit.json missing/invalid fields
CI_FAILED          -> required tests did not pass
POLICY_VIOLATION   -> violated safety/tool/scope rules
EXECUTOR_ERROR     -> tool/runtime error prevented completion
TIMEOUT_EXCEEDED   -> exceeded time limit
BUDGET_EXCEEDED    -> exceeded token/step budget
NEED_INPUT         -> blocked; user action/clarification required

== REQUIRED ARTIFACTS (5) ==
1. artifacts/report.md
2. artifacts/selftest.log
3. artifacts/evidence/
4. artifacts/patch.diff
5. artifacts/submit.json

[END IO DIGEST]
